633\. Sum of Square Numbers

Medium

Given a non-negative integer `c`, decide whether there're two integers `a` and `b` such that <code>a<sup>2</sup> + b<sup>2</sup> = c</code>.

**Example 1:**

**Input:** c = 5

**Output:** true

**Explanation:** 1 \* 1 + 2 \* 2 = 5

**Example 2:**

**Input:** c = 3

**Output:** false

**Constraints:**

*   <code>0 <= c <= 2<sup>31</sup> - 1</code>
